Mostrar y Ocultar Texto

Haz clic en el botón para mostrar u ocultar el texto:

Este es el texto que aparecerá y desaparecerá cuando hagas clic en el botón.

Código JavaScript utilizado:

function toggleTexto() {
    var texto = document.getElementById("miTexto");
    if (texto.classList.contains("hidden")) {
        texto.classList.remove("hidden");
    } else {
        texto.classList.add("hidden");
    }
}
